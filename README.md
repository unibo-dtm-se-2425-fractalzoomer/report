# Fractal Zoomer - Project Report

This repository contains the full documentation for the Fractal Zoomer project, developed as part of the Software Engineering course (2024-2025) at the University of Bologna.

## About the Project

Fractal Zoomer is an interactive desktop application for exploring mathematical fractals. Built with Python, NumPy, and tkinter, it lets you zoom, pan, and explore three different fractals: Mandelbrot set, Julia sets, and Burning Ship.

## Quick Start

To run the application:

```bash
git clone https://github.com/unibo-dtm-se-2425-fractalzoomer/artifact.git
cd artifact
poetry install
poetry run fractalzoomer
```
